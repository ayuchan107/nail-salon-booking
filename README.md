# サロン予約システム

Next.js 14とTailwind CSSで構築されたサロン向けの予約管理システムです。

## 特徴

- 💅 **美しいUI** - サロンにふさわしいピンク・パープル系のデザイン
- 📅 **2ヶ月間の予約表示** - 今日から2ヶ月分の予約枠を週間表示
- 🗓️ **週間ナビゲーション** - 前週・次週ボタンで簡単に日付選択
- ⭕❌ **直感的な表示** - 予約可能は⭕、予約済みは❌で分かりやすく表示
- 👤 **お客様モード** - 予約可能な時間帯をクリックして予約
- 👩‍💼 **管理者モード** - ワンクリックで予約状況の変更が可能
- 🍽️ **メニュー管理** - メニューの追加・編集・削除が可能
- 💾 **データ永続化** - ローカルストレージで予約データを保存
- 📱 **レスポンシブ対応** - スマートフォンでも使いやすい

## 営業時間・予約枠

- **営業時間**: 10:00 - 23:00（最終受付22:00）
- **予約間隔**: 1時間ごと
- **表示期間**: 今日から2ヶ月間（週間表示で切り替え）

## 使い方

### お客様の予約手順
1. 希望の日付・時間の⭕をクリック
2. メニューを選択（ベーシック、ジェル、アート、ケア、リペア）
3. お名前と電話番号を入力
4. 「予約確定」ボタンで予約完了
### 管理者の操作手順
1. 「管理者モード」ボタンをクリックしてONに切り替え
2. 「予約管理」ボタンで管理画面を表示
3. 「メニュー管理」ボタンでメニュー編集画面を表示
4. 今日の予約と予約履歴を確認
5. 過去の予約をクリックして施術記録を入力
6. スケジュール画面で⭕/❌をクリックして予約状況を手動変更

### メニュー管理機能
1. 管理者モードで「メニュー管理」をクリック
2. メニュー名、施術時間、料金を入力して「メニュー追加」
3. 既存メニューの「編集」ボタンで内容変更
4. 不要なメニューは「削除」ボタンで削除
5. 変更は即座に予約フォームに反映
5. スケジュール画面で⭕/❌をクリックして予約状況を手動変更

## 技術スタック

- **Next.js 14** - React フレームワーク（App Router使用）
- **TypeScript** - 型安全性のための言語
- **Tailwind CSS** - ユーティリティファーストのCSSフレームワーク
- **ESLint** - コード品質の維持

## 開発環境のセットアップ

1. 依存関係のインストール:
```bash
npm install
```

2. 開発サーバーの起動:
```bash
npm run dev
```

3. ブラウザで [http://localhost:3000](http://localhost:3000) を開く

## 利用可能なスクリプト

- `npm run dev` - 開発サーバーの起動
- `npm run build` - プロダクション用ビルドの作成
- `npm run start` - プロダクション用サーバーの起動
- `npm run lint` - ESLintによるコードチェック

## カスタマイズ

### 営業時間の変更
`src/app/page.tsx`の`generateInitialSchedule`関数内の`timeSlots`配列を編集してください。

```typescript
const timeSlots = [
  '10:00', '11:00', '12:00', '13:00', '14:00', '15:00',
  '16:00', '17:00', '18:00', '19:00', '20:00', '21:00', '22:00', '23:00'
]
```

### 表示期間の変更
`generateInitialSchedule`関数内のループ条件を変更してください。

```typescript
for (let i = 0; i < 60; i++) { // 60を変更して期間を調整（日数）
```
